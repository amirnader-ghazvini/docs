## Contracts & Addresses

## Upgradeable OFTs

### EVM

#### Semi-Deterministic Addresses

Each of the 6 OFTs maintain the same address across these chains:

- Mode
- Sei
- X-Layer
- Ink
- Sonic
- Arbitrum
- Optimism
- Polygon
- Avalanche
- BSC
- Polygon zkEVM
- Berachain
- Blast
- Worldchain

#### Random Addresses

Each address of these OFTs is unique:

- Linea, Base
  - All 6 OFTs
- Ethereum
  - WFRAX OFT

#### zkSync

- zkSync Era, Abstract maintain the same addresses across all 6 OFTs. This is due to the ZK compiler.
